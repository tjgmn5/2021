Relational Database Service (RDS)

RDS (or Relational Database Service) is a service that aids in the administration and management of databases. RDS assists with database administrative tasks that include upgrades, patching, installs, backups, monitoring, performance checks, security, etc.

Database Engine Support

  • Oracle
  • PostgreSQL
  • MySQL
  • MariaDB
  • SQL Server

Features

  • failover
  • backups
  • restore
  • encryption
  • security
  • monitoring
  • data replication
  • scalability

Which database engines are supported by RDS? Pick three.
SOLUTION:
  • MySQL
  • Aurora
  • Oracle

What doesn't RDS help with?
SOLUTION: Accessing your database via secure shell (SSH)
