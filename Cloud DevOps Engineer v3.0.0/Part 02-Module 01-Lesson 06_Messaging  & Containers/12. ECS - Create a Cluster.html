12. ECS - Create a Cluster

In this hands-on exercise, you will learn how to create a Cluster in Amazon ECS.

  • Prerequisites: AWS Account

Follow these steps to create a Cluster in ECS.

  1. Navigate to the ECS dashboard, and select the Clusters service from the left-hand side menu pane.

  2. On the Clusters page, choose the Create cluster button.

  1. At the Select cluster template step, choose the Networking only compatibility.

  1. At the Configure Cluster step, enter the name for your cluster. Choose a unique name.

  2. In the Networking section, create a new VPC for your cluster. You can keep the default settings for the subnet division.

  1. Enable the CloudWatch container insights. And finally, click the Create button.

Congratulations, you’ve created your first Cluster
