Checking your Understanding

Now let’s review your knowledge on a couple of AWS items.

AWS Setup Q1

What is the most expensive cost in AWS?
SOLUTION: EC2 instances

AWS Setup Q2

What does IAM stand for?
SOLUTION: Identity and Access Management

Billing MCQ

Select all that is True related to AWS Billing for free tier accounts.
SOLUTION:
  • Services such as Amazon EC2, Amazon Cloudwatch, Amazon RDS, and Elastic Load Balancing, charge for usage on an hourly basis.
  • The free tier allocation for Linux and Microsoft Windows instances is calculated independently.
  • For EC2 limits under free tier, you can operate (active state) 750 hours of a Linux t2.micro or t1.micro instance AND 750 hours of a Windows t2.micro / t1.micro instance each month for the first year.
