12. Jenkins Quiz

Radio Quiz

What language is Jenkins written in?
SOLUTION: Java

MCQ

What is Jenkinsfile?
SOLUTION:
  • A text file that contains the definition of a Jenkins Pipeline.
  • A file that can be written in either of two types of syntax - Declarative versus Scripted.
  • A configuration file that can define steps to be performed in different stages through the specified agent.
