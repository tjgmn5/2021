Greedy Algorithms

This change example uses a “greedy algorithm”. Greedy algorithms use a heuristic of finding the locally optimal choice to solve a global optimum. There are many examples of greedy algorithms we use in the real world. A reasonable solution to a famous optimization problem, The Traveling Salesman problem, can be found by randomly starting at a route, and then always picking the shortest distance to the next city.
